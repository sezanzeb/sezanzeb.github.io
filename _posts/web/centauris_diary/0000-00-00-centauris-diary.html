--- layout: post excerpt_separator: "" title: "Centauri's Diary" date: "2019-07-18 02:00:00 -0000" --- Very minimal website software that works like a blog. Written in php and is running on this website.
  • Controlled from the file system by using .html and .json files, no admin backend
  • Can have blogs with articles, organized by categories, or static pages
  • A Template file is used to controll the page's layout
  • Generates navigations and table of contents automatically
  • That's it. Runs on your apache php server
